# trend-pulse-privacy
Privacy policy for TrendPulse research tool
                                                                                                                                                        
     Privacy Policy                                                                                                                                                    
                                                                                                                                                                       
     TrendPulse — Trend Intelligence Research Tool                                                                                                                     
     Last updated: 2026-01-06                                                                                                                                          
     Information We Collect                                                                                                                                            
                                                                                                                                                                       
     TrendPulse is a non-commercial trend monitoring and research tool. We collect publicly available content metadata (pins, descriptions, images, hashtags) from     
     Pinterest via the official Pinterest API for the sole purpose of fashion and lifestyle trend analysis.                                                            
                                                                                                                                                                       
     We do NOT collect:                                                                                                                                                
     - Personal information of Pinterest users                                                                                                                         
     - Private boards or non-public content                                                                                                                            
     - User login credentials or passwords                                                                                                                             
     - Location data or device identifiers                                                                                                                             
     How We Use Information                                                                                                                                            
                                                                                                                                                                       
     Collected data is used exclusively for:                                                                                                                           
     - Trend pattern analysis and visualization                                                                                                                        
     - Fashion and lifestyle market research                                                                                                                           
     - Aggregated trend reporting                                                                                                                                      
                                                                                                                                                                       
     All analysis is performed on aggregated, anonymized data. Individual pins are processed as data points within larger trend datasets.                              
     Data Storage and Security                                                                                                                                         
                                                                                                                                                                       
     - API access tokens are stored securely on encrypted cloud storage                                                                                                
     - Collected metadata is stored in private cloud object storage (MinIO)                                                                                            
     - Data retention: 30 days, after which it is automatically purged                                                                                                 
     - No data is shared with third parties
